# Crossing Cube

This is a very old game that I've decided to put on GitHub. It's even older than [Cube.](https://github.com/jmxjiang/cube) The turtle's name is Cube, and you have to cross the rectangles. To move up, press W on the keyboard (sorry if you're on mobile). And yes, you have to repeatedly press W; you can't hold it (I told you this game is old). I don't think level 10 is beatable.
